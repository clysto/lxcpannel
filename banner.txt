Welcome to the LXC Panel v1.0.0

 * ip:                            show the current ip address
 * whoami:                        show the current user
 * lxc list:                      list all containers
 * lxc images:                    list all images
 * lxc start  <name>:             start a container
 * lxc stop   <name>:             stop a container
 * lxc create <name>:             create a container
 * lxc shell  <name>:             open a shell in a container
 * lxc delete <name>:             delete a container
 * pubkey list:                   list all public keys
 * pubkey add <key>:              add a public key
 * pubkey delete <fingerprint>:   delete a public key
